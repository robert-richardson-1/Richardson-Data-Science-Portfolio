## Description

This Streamlit app explores the "Palmer's Penguins" Dataset. Using the added filters, one can search and organize the penguins' data by the following:
- species
- island
- bill length (mm)
- bill depth (mm)
- flipper length (mm)
- body mass (g)
- sex
- year
